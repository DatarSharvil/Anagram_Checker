# About:
  A CLI tool that checks Anagrams in CLI on Linux x64 (Debian based and non-Debian based) and Windows 8+ x64 without an internet. Enjoy!
# Requirements:
  <p> Minimum: Java 8 </p>
  <p> Recommended: Java 17 </p>

# Usage:
  1. Linux (Debian-based): Just install the .deb file the way you like and enter the command "anagram" from any terminal to execute.
  2. Linux (Non-Debian-based): Extract the tar.gz file and run the script.
  3. Windows: Double click the .exe to run or add it to the PATH to run it from anywhere.
